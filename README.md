1.冒泡排序：
	时间复杂度：N + N-1 + N - 2 + .... + 1
	        = N(1 + N)/2
	        = N + N^2/2
	        = N^2
	        = big O(1) * N^2
	        = O(N^2)

			Sn = n(a1 + an)/2
			   = n*a1 + n(n-1)d/2
			   
	最好情况时间复杂度：O(n)
	最坏情况时间复杂度：O(n^2)
